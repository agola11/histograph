This page helps you filter to URLs which you want to be used by Histograph

The following graph shows all top-level domains which you have visited (e.g. www.google.com)

  • Click on a bubble to view sub-domains (e.g. www.google.com/maps)
  • Shift-Click on a bubble to block that domain or sub-domain and all sub-domains from being used by Histograph.
  • Ctrl-Shift-Click on a bubble to block a sub-domain and all sites visited from links on that site.
  • Ctrl-Click on a bubble to unblock that sub-domain.
